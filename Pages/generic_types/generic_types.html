Generic Types

Generics provide a way to create reusable components that can work with a variety of types rather than a single one. They allow you to define a function, class, or interface with a placeholder for the type, which can be specified when the component is used.

Generic Functions

Here's an example of a generic function that takes an array of items and returns the first item:

function getFirstItem<T>(items: T[]): T {
    return items[0];
}
const numbers = [1, 2, 3];
const firstNumber = getFirstItem(numbers); // firstNumber is of type number
const strings = ["a", "b", "c"];
const firstString = getFirstItem(strings); // firstString is of type string

Generic Classes

You can also create generic classes. Here's an example of a generic class that implements a simple stack data structure:

class Stack<T> {
    private items: T[] = [];
    push(item: T): void {
        this.items.push(item);
    }
    pop(): T | undefined {
        return this.items.pop();
    }
}
const numberStack = new Stack<number>();
numberStack.push(1);
numberStack.push(2);
const poppedNumber = numberStack.pop(); // poppedNumber is of type number
const stringStack = new Stack<string>();
stringStack.push("a");
stringStack.push("b");
const poppedString = stringStack.pop(); // poppedString is of type string

Generic Interfaces

Generics can also be used with interfaces. Here's an example of a generic interface for a key-value pair:

interface KeyValuePair<K, V> {
    key: K;
    value: V;
}
const pair: KeyValuePair<string, number> = {
    key: "age",
    value: 30
};

In this example, the KeyValuePair interface uses two type parameters, K and V, to represent the types of the key and value, respectively.
